# SKN_Pjt04

# 💻문서 Q&A 웹 서비스

본 프로젝트는 AI 기반 문서 Q&A 시스템을 웹 서비스로 구현한 것입니다.

Django 웹 프레임워크를 활용하여 사용자 친화적인 인터페이스를 제공하고, 직관적인 문서 업로드 및 검색 기능을 구현했습니다.
